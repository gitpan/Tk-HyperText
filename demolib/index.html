Tk::HyperText Demonstration

The program you are viewing right now is the demonstration program shipped with the Perl module, Tk::HyperText. The demo acts as a kind of simple "web browser", reading and rendering the pages from the `demolib` folder.

For more demonstrations, please follow one of the links below. You can return here by pressing the "Home" button at the top of your screen.

Note: Pressing Ctrl+S will display the HTML source code of the current page in your terminal.
